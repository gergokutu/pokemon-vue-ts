# pokemon-vue-ts

Browse information about Pokemons in the application. Data fetched with axios from pokeapi.co...

- browse Pokemons
- check Pokemon details
- get random details
- see all the existing types of Pokemons

## Technologies

- Vue / Vuex
- BootstrapVue
- TypeScript
- Axios
- HTML / CSS
